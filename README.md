# ⚖️ Islam Falsely Accused — الإسلام المفترى عليه

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **الإسلام المفترى عليه (Islam Falsely Accused)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards addressing common accusations against Islam and their refutations
- Topics: violence, women's rights, science, freedom, tolerance, modernity
- Daily habit tracker, knowledge quiz, selected duas
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 themes, Narrator, Responsive + PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.
